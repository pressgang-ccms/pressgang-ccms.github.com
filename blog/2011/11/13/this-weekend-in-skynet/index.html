This Weekend in Skynet

Nov 13th, 2011

A busy weekend has added grouping to search results.

Grouped Topic Searches

Search results can now be grouped. The main search screen has a new column next to the tags labelled Grp. Any tag with the Grp (or grouping) checkbox selected will be used to group the search result.

On the search screen each grouping tag is represented by a tab. These tabs list all the topics that match the filter AND include the grouping tag. This means that a topic might be listed more than once under different tabs.

Topics that don’t include any of the grouping tags will be listed under a tab called Ungrouped Results. This means that, when using grouping, all topics that would have been returned by the filter are listed at least once under once of the tabs, and possibly more than once.

All group operations, such as bulk tagging or group relationship modifications, are performed on all the topics, and not just those in the currently selected tab.

Exception Logging

All exceptions are now logged to the database. This should allow me to monitor occurrences of the angry terminator, and hopefully resolve these kinds of issues in a more timely fashion.
