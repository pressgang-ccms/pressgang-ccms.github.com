PressGang CCMS Source

The PressGang CCMS source code is hosted on github.com under the pressgang-ccms organisation, https://github.com/pressgang-ccms/.

The projects are:

  • afterthedeadline-chrome

    Last update: 29/11/2012.

  • afterthedeadline-openshift

    Last update: 29/11/2012.

  • j2bugzilla-extended

    Last update: 14/11/2012.

  • j2koji

    This project provides a mechanism for interacting with a Koji/Brew installation using it’s XML-RPC API.

    Last update: 26/11/2012.

  • pressgang-ccms.github.com

    http://pressgang-ccms.github.com blog for updates and docs

    Last update: 02/12/2012.

  • pressgang-parent

    Last update: 13/11/2012.

  • PressGangCCMSCommonUtilities

    This library provides a number of common methods for things such as manipulating collections, working with XML, working with files, working with threads and more. Although the library is used by the PressGang CCMS product, this library is designed to used with other projects.

    Last update: 27/11/2012.

  • PressGangCCMSContentSpec

    Last update: 14/11/2012.

  • PressGangCCMSModel

    This project supplies the POJO entity and basic utility classes for the PressGang Database Mapping. The project uses Hibernate as the JPA 2.0 implementation for Object-Relational Mapping.

    Last update: 27/11/2012.

  • PressGangCCMSREST

    The PressGang CCMS REST Server project provides a RESTful web interface for the PressGang CCMS (Component Content Management System). It offers create, update, get and delete endpoints for the system’s primary entities. These endpoints produce both JSON and JSONP, which adds cross-domain functionality. Topic entities also have ATOM, XML and HTML get endpoints.

    Last update: 27/11/2012.

  • PressGangCCMSRESTCommon

    Last update: 13/11/2012.

  • PressGangCCMSRESTv1Common

    The JAX-RS interface and POJO entities for version 1 of the PressGang CCMS REST interface

    Last update: 27/11/2012.

  • PressGangCCMSServices

    This project contains all of the external services that can be used in a PressGang CCMS instance.

    Last update: 29/11/2012.

  • PressGangCCMSUI

    PressGang CCMS UI

    Last update: 26/11/2012.

  • PressGangCCMSZanataInterface

    This project provides an interface to the Zanata translation platform. See http://zanata.org/ for more information.

    Last update: 26/11/2012.

These are forks of projects used by PressGang CCMS:

  • AceGWT

    Integrating ACE (Ajax.org Code Editor) with GWT

    Last update: 15/11/2012.

  • gozirra-extended

    stomp java library

    Last update: 14/11/2012.

The PressGang CCMS

PressGang CCMS is a web-based Component Content Management System (CCMS) written in Java.

It is open source and developed on JBoss Application Server 7.
